A quarter is divided into small communities called homes. These are personal social networks for local groups, designed as an alternative to Facebook chats and groups.

John 14:05

Thanks! Homes can be created by anyone, on any server running the Quarter software, and can be joined by whoever knows the group's web address.

David 14:12

What about privacy? How can I trust the operator of the server not to read my posts?

John 14:16

Actually, that's the great thing about Quarter: you don't have to trust the server operator. All of your posts are encrypted in your web browser and only then uploaded to the server. This means that the content of your posts is unreadable to the server operator.

David 14:21

Encryption – does that mean that I need to remember some sort of key to unlock the home?

John 14:16

That's another great thing about Quarter :-) The encryption key is contained within the home's address – in fact, a home's key is the same as its name.

In the address https://quarter.group/#/my-own-home, the key is "my-own-home", but because this part of the address is hidden behind a # symbol, it is never transmitted to the server.

Instead, it is used to unlock the home directly in your web browser, via JavaScript.
